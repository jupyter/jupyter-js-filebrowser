API Docs

Package Install

Prerequisites

  • node
npm install --save jupyter-js-filebrowser

Source Build

Prerequisites

  • git
  • node 0.12+
git clone https://github.com/jupyter/jupyter-js-filebrowser.git
cd jupyter-js-filebrowser
npm install
npm run build

Rebuild

npm run clean
npm run build

Run Tests

Follow the source build instructions first.

npm test

Build Docs

Follow the source build instructions first.

npm run docs

Navigate to docs/index.html.

Build Example

Follow the source build instructions first. Requires a Python install with the Jupyter notebook 4.1+.

npm run build:example

Run python main.py from the example directory.

Supported Runtimes

The runtime versions which are currently known to work are listed below. Earlier versions may also work, but come with no guarantees.

  • IE 11+
  • Firefox 32+
  • Chrome 38+

Bundle for the Browser

Follow the package install instructions first.

Any bundler that understands how to require() files with .js and .css extensions can be used with this package.
